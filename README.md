## Instructions

This project is write with the 42's norminette.

The functions should not quit unexpectedly.

All heap-allocated memory space must be properly freed when necessary.

Create a Makefile that compiles all the sources files to the required output with the flags -Wall, -Wextra, and -Werror, using cc, and the Makefile must not relink.

The Makefile must at least contain the rules $(NAME), all, clean, fclean, and re.
